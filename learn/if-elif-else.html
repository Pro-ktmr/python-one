if 文の拡張

if 文では if 条件式: に続けて、条件に合わなかった場合の処理を追加することができます。条件に合わなかった場合の中で、さらに別の条件を判定したい場合は elif 条件式: を使います。また、条件に合わなかった場合に無条件で実行したい処理がある場合は else: を使います。
