基本のプログラム①　出力（print(計算式)）

プログラムとは、コンピュータに対する指示書のようなものです。上から順番に 111 行ずつ、コンピュータは指示を読み取って実行していきます。

基本ルール

Python でプログラムを書くときに守らないといけない基本ルールが 222 つあります。最初に確認しておきましょう。

  • 基本的にすべての文字を半角文字で書きます。半角文字というのは、a や 1 のような縦長の文字のことです。それに対し、全角文字というのは、ａ や １ のような真四角な文字のことです。日本語（ひらがな）を入力できる状態になっていると、全角文字が入力されるようになってしまいます。Windows を使っている人はキーボードの左上にある「半角／全角」キーを押して、半角文字が入力される状態にしてからプログラムを書き始めましょう。
  • 基本的にすべての文字を小文字で書きます。小文字と大文字は別の文字として扱われるので、大文字で書くとエラーになってしまいます。

出力

プログラムの実行結果を画面に表示することを出力と言います。値を出力したいときは print( 計算式 ) というように、print を使って書きます。括弧の中に出力したい値や計算式を書きます。
