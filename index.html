PythonOne ―― 高校情報ⅠのためのPythonオンライン学習環境

PythonOne は高等学校の科目「情報Ⅰ」のためのPythonオンライン学習環境です。各出版社の「情報Ⅰ」の教科書に準拠して学習コンテンツを作成しています。

学習コンテンツ一覧

基本のプログラム

  1. 出力（print(計算式)）
  2. 四則演算（+ - * / // %）
  3. 変数（変数名 = 計算式）
  4. 入力（input()）
  5. 文字列（" '）

条件分岐

  1. 単一の条件分岐（if 条件式:）・比較演算（== != < > <= >=）
  2. 複数の条件分岐（if 条件式: elif 条件式: else:）
  3. 論理演算（and or not）

繰り返し

  1. 回数を指定する繰り返し（for 変数名 in range(繰り返し回数):）
  2. 条件を指定する繰り返し（while 条件式:）

リスト

  1. リスト（[ ]）
  2. リストと繰り返し（合計値・最小値・最大値・探索・数え上げ・全部）

関数

  1. 組み込み関数の呼び出し（組み込み関数の呼び出し（sum() min() max()） ）
  2. 関数の定義（def 関数名(引数):）

外部リンク集

  • プログラムを自由に書くなら PyTry
  • 対話式のプログラムを自由に書くなら PyTry（インタラクティブモード）
